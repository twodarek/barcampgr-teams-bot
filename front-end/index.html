How Does This Work?

We've got a bot, and you can talk to it in the Cisco Teams space!

Here's what it can do:

  • Send you a link to a web form where you can schedule a talk – To do this, type @barcampgrbot schedule web into the chat in any room
  • Schedule a talk for you – To do this, type @barcampgrbot schedule me at START_TIME in ROOM for TITLE into the chat in any room
  • Send you the unique edit links to any talks you've scheduled – To do this, type @barcampgrbot get links into the chat in any room
  • For a full list of what the bot can do, type @barcampgrbot help into the chat in any room
